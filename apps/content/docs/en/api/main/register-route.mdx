---
title: registerRoute
description: Registers a window as a route of the application.
---

The `registerRoute` method returned by the <Link href="/docs/api/create-electron-router">createElectronRouter</Link> function registers a window as a route of the application.
This means that only the content defined from the corresponding route in the renderer process for that window will be displayed in it.

## Basic Example

```ts
  registerRoute({
    id: 'main',
    browserWindow: mainWindow,
    htmlFile: path.join(__dirname, '../renderer/index.html'),
  })
```
