O componente `Router` retornado pela função <Link href="/docs/api/create-electron-router">createElectronRouter</Link> é responsável por gerenciar as rotas a partir das janelas correspondentes no processo de renderização.

Internamente, ele usa o `RouterProvider` e a função `createHashRouter` do pacote `react-router-dom` e aplica a lógica necessária para que as janelas correspondentes sejam exibidas corretamente.

## Exemplo básico

```tsx
  <Router
    main={
      <Route path="/" element={<Layout />} errorElement={<ErrorScreen />}>
        <Route
          path="/"
          element={<MainScreen />}
        />

        <Route path="/about" element={<AboutScreen />} />
      </Route>
    }
  />
```
